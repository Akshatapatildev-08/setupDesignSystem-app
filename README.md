# Job Notification App - Design System Foundation

This foundation contains only the design system shell and primitives.
No product feature workflows are included.

## Included
- Global structure: Top Bar, Context Header, 70/30 Workspace split, Proof Footer
- Color tokens constrained to 4-color palette
- Success (muted green) and warning (muted amber) semantic tokens defined for state messaging
- Semantic states constrained by the same 4-color cap (no extra state-only colors)
- Typography with serif heading and clean sans-serif body
- Spacing tokens limited to 8, 16, 24, 40, 64
- Component primitives: cards, buttons, inputs, status badges, state messaging
- Motion baseline: 180ms ease-in-out transitions

## Files
- `index.html`: layout shell and component examples
- `styles.css`: design tokens and styling rules
